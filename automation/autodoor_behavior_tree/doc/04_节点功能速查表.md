# AutoDoor 行为树系统 — 节点功能总表

本文档提供所有节点的快速参考，包含所有参数及详细说明。

***

## 复合节点（5个）

### 1. 开始节点 (StartNode)

| 参数名 | 说明 | 类型 | 默认值 | 显示条件 |
| --- | --- | --- | --- | --- |
| 绑定窗口 | 是否绑定窗口 | bool | False | - |
| 窗口标题 | 绑定窗口的标题 | window_select | 无 | 绑定窗口=True |
| 窗口PID | 绑定窗口的进程ID（隐藏字段） | number | 0 | 隐藏 |

> 注：开启"绑定窗口"后，所有坐标将自动转换为窗口相对坐标；"窗口PID"为隐藏字段，自动获取

### 2. 顺序节点 (SequenceNode)

| 参数名 | 说明 | 类型 | 默认值 | 显示条件 |
| --- | --- | --- | --- | --- |
| 子节点间隔(ms) | 子节点执行间隔 | number | 0 | - |
| 子节点间隔随机范围(±ms) | 随机抖动 | number | 0 | - |
| 失败后继续执行 | 开启后会先把子节点都运行后，再根据运行情况返回成功/失败 | bool | False | - |

### 3. 选择节点 (SelectorNode)

| 参数名 | 说明 | 类型 | 默认值 | 显示条件 |
| --- | --- | --- | --- | --- |
| 子节点间隔(ms) | 子节点执行间隔 | number | 0 | - |
| 子节点间隔随机范围(±ms) | 随机抖动 | number | 0 | - |

### 4. 并行节点 (ParallelNode)

| 参数名 | 说明 | 类型 | 默认值 | 显示条件 |
| --- | --- | --- | --- | --- |
| 成功策略 | 决定节点成功条件 | select | require_all | - |

**成功策略选项：**
- require_all：全部子节点成功才成功
- require_one：任一子节点成功即成功

### 5. 随机节点 (RandomNode)

| 参数名 | 说明 | 类型 | 默认值 | 显示条件 |
| --- | --- | --- | --- | --- |
| 成功策略 | 决定节点成功条件 | select | require_all | - |
| 失败后继续执行 | 开启后继续执行 | bool | False | - |
| 完全随机 | 每次随机选择 | bool | False | - |

**成功策略选项：**
- require_all：全部子节点成功才成功
- require_one：任一子节点成功即成功

## 条件节点（6个）

### 1. OCR检测节点 (OCRConditionNode)

| 参数名 | 说明 | 类型 | 默认值 | 显示条件 |
| --- | --- | --- | --- | --- |
| 检测区域 | 截图区域 (x1,y1,x2,y2) | region | 无 | - |
| 关键词 | 要检测的文字，多个用逗号分隔 | text | 无 | - |
| 语言 | OCR识别语言 | select | 简体中文 | - |
| 图像预处理 | 图像预处理模式 | select | 默认 | - |
| 位置变量名 | 保存位置的黑板变量名 | text | last_detection_position | - |
| 坐标偏移 | 相对于检测位置的偏移 (x,y) | offset | (0,0) | - |

**语言选项：** 简体中文、English、繁体中文
**预处理模式说明：**

| 模式 | 说明 | 适用场景 |
| --- | --- | --- |
| 默认 | 标准预处理，固定阈值二值化 | 普通文本、高对比度图像 |
| 复杂色彩 | 放大+锐化+固定阈值 | 游戏界面、复杂背景 |
| 自适应 | CLAHE+自适应二值化 | 低对比度、渐变背景 |
| 自动调优 | 根据图像特征自动选择参数 | 所有场景，推荐使用 |

### 2. 图像匹配节点 (ImageConditionNode)

| 参数名 | 说明 | 类型 | 默认值 | 显示条件 |
| --- | --- | --- | --- | --- |
| 检测区域 | 截图区域 | region | 无 | - |
| 模板路径 | 匹配图像文件路径 | screenshot | 无 | - |
| 匹配阈值(%) | 匹配度阈值 0-100 | number | 80 | - |
| 位置变量名 | 保存位置的黑板变量名 | text | last_detection_position | - |
| 坐标偏移 | 相对于检测位置的偏移 | offset | (0,0) | - |

> 注：模板路径支持截图导入和文件浏览两种方式

### 3. 颜色检测节点 (ColorConditionNode)

| 参数名 | 说明 | 类型 | 默认值 | 显示条件 |
| --- | --- | --- | --- | --- |
| 检测区域 | 截图区域 | region | 无 | - |
| 目标颜色 | #RRGGBB格式颜色值 | color | 无 | - |
| 容差 | 颜色容差 0-100 | number | 10 | - |
| 最小像素数 | 最少匹配像素数 | number | 1 | - |
| 位置变量名 | 保存位置的黑板变量名 | text | last_detection_position | - |
| 坐标偏移 | 相对于检测位置的偏移 | offset | (0,0) | - |

> 注：目标颜色支持颜色选择器和手动输入RGB值

### 4. 数字比较节点 (NumberConditionNode)

| 参数名 | 说明 | 类型 | 默认值 | 显示条件 |
| --- | --- | --- | --- | --- |
| 检测区域 | 截图区域 | region | 无 | - |
| 预处理模式 | 图像预处理模式 | select | 普通文本 | - |
| 提取模式 | 数字提取方式 | select | 无规则 | - |
| 自定义模式 | 自定义正则提取 | text | 无 | 提取模式=自定义 |
| 比较模式 | 比较运算符 | select | == | - |
| 比较值 | 要比较的数值 | number | 0 | - |
| 置信度阈值(%) | 识别置信度阈值 | number | 50 | - |
| 值变量名 | 保存数值到黑板变量 | text | last_number_value | - |
| 位置变量名 | 保存位置的黑板变量名 | text | last_detection_position | - |
| 坐标偏移 | 相对于检测位置的偏移 | offset | (0,0) | - |

**提取模式选项：** 无规则、x/y、自定义（选择自定义时显示"自定义模式"参数）

### 5. 变量判断节点 (VariableConditionNode)

| 参数名 | 说明 | 类型 | 默认值 | 显示条件 |
| --- | --- | --- | --- | --- |
| 变量名 | 要判断的黑板变量名 | text | 无 | - |
| 运算符 | 比较运算符 | select | == | - |
| 比较值 | 要比较的值 | text | 无 | 运算符≠exists/not_exists |

**运算符选项：**
- ==, !=, >, <, >=, <=
- exists（变量存在）, not_exists（变量不存在）
- contains（包含）, not_contains（不包含）
- starts_with（开头是）, ends_with（结尾是）

> 注：选择 exists 或 not_exists 时，"比较值"参数隐藏

### 6. 文本提取节点 (TextExtractNode)

| 参数名 | 说明 | 类型 | 默认值 | 显示条件 |
| --- | --- | --- | --- | --- |
| 检测区域 | 截图区域 | region | 无 | - |
| 提取模式 | 提取方式 | select | all | - |
| 关键词 | 要提取的关键词 | text | 无 | 提取模式=keywords |
| 语言 | OCR识别语言 | select | 简体中文 | - |
| 图像预处理 | 预处理模式 | select | 默认 | - |
| 输出变量名 | 保存提取文本的变量名 | text | last_extracted_text | - |
| 保存全部文本 | 是否保存所有识别文本 | bool | False | - |
| 全部文本变量名 | 保存全部文本的变量名 | text | all_ocr_text | 保存全部文本=True |
| 保存位置 | 是否保存检测位置 | bool | True | - |
| 位置变量名 | 保存位置的黑板变量名 | text | last_detection_position | - |
| 坐标偏移 | 相对于检测位置的偏移 | offset | (0,0) | - |

**提取模式选项：** all(提取全部)、keywords(提取关键词)
**预处理模式说明：**

| 模式 | 说明 | 适用场景 |
| --- | --- | --- |
| 默认 | 标准预处理，固定阈值二值化 | 普通文本、高对比度图像 |
| 复杂色彩 | 放大+锐化+固定阈值 | 游戏界面、复杂背景 |
| 自适应 | CLAHE+自适应二值化 | 低对比度、渐变背景 |
| 自动调优 | 根据图像特征自动选择参数 | 所有场景，推荐使用 |

> 注：选择"提取关键词"模式时显示"关键词"参数；开启"保存全部文本"时显示"全部文本变量名"参数

## 黑板系统

### 内置变量

| 变量名 | 说明 |
| --- | --- |
| last_detection_position | 最近一次检测到的位置坐标 (x, y) |
| last_number_value | 最近一次识别的数字值 |
| last_extracted_text | 最近一次提取的文本 |
| last_input_text | 最近一次输入的文本 |

### 自定义位置变量

多个检测节点支持自定义位置保存的变量名（position_key参数），默认为 `last_detection_position`，可用于多检测点场景。

***

## 节点状态

| 状态 | 说明 |
| --- | --- |
| 成功 | 节点执行成功 |
| 失败 | 节点执行失败 |
| 运行中 | 节点正在执行，需要继续tick |

***

## 常见用途速查

| 用途 | 推荐节点组合 |
| --- | --- |
| 点击按钮 | 图像匹配节点 → 鼠标点击节点 |
| 检测文字并点击 | OCR检测节点 → 鼠标点击节点 |
| 循环执行 | 顺序节点 + 变量判断节点 + 设置变量节点 |
| 多方案尝试 | 选择节点 + 多个条件节点 |
| 并行检测 | 并行节点 + 多个条件节点 |
| 等待加载 | 延时节点 |
| 数值判断 | 数字比较节点 |
| 状态控制 | 变量判断节点 + 设置变量节点 |
| 文本输入 | 文本提取节点 → 文本输入节点 |
